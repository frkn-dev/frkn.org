Как подключиться

Выбери свою платформу и следуй инструкции. Всё настроится за пару минут — достаточно установить приложение и вставить ссылку подписки.

iOS

iPhone и iPad

Streisand
Бесплатно Рекомендуем
Скачать в App Store
  1. Установи Streisand из App Store
  2. Открой приложение и нажми Import
  3. Вставь ссылку подписки из личного кабинета
  4. Выбери сервер и нажми Connect
Shadowrocket
Платное
Скачать в App Store
  1. Купи и установи Shadowrocket
  2. Нажми плюсик в правом верхнем углу
  3. Вставь ссылку подписки
  4. Выбери сервер и подключись

Android

Смартфоны и планшеты

v2rayNG
Бесплатно Рекомендуем
Скачать с GitHub
  1. Скачай и установи APK с GitHub
  2. Открой приложение и нажми +
  3. Выбери Import from clipboard
  4. Вставь ссылку подписки
  5. Выбери сервер и нажми значок подключения
Hiddify
Бесплатно
Перейти на сайт
  1. Установи Hiddify из GitHub или Google Play
  2. Нажми Добавить подписку
  3. Вставь ссылку подписки
  4. Нажми подключение

Windows

ПК и ноутбуки

v2rayN
Бесплатно Рекомендуем
Скачать с GitHub
  1. Скачай и распакуй архив с GitHub
  2. Запусти v2rayN.exe
  3. Нажми Servers → Import from clipboard
  4. Вставь ссылку подписки
  5. Выбери сервер и нажми правой кнопкой → Set as active server
Hiddify
Бесплатно
Перейти на сайт
  1. Скачай и установи Hiddify
  2. Нажми Добавить подписку
  3. Вставь ссылку подписки
  4. Выбери режим VPN или прокси

macOS

MacBook и iMac

V2RayXS
Бесплатно Рекомендуем
Скачать с GitHub
  1. Скачай приложение и перетащи в Applications
  2. Запусти и открой меню из строки состояния
  3. Нажми Import from clipboard
  4. Вставь ссылку подписки
  5. Выбери сервер и включи прокси
Clash Verge
Бесплатно
Скачать с GitHub
  1. Скачай и установи Clash Verge
  2. Импортируй ссылку подписки
  3. Выбери режим System Proxy или TUN
  4. Нажми кнопку подключения
Shadowrocket
Платное
Скачать в App Store
  1. Купи и установи Shadowrocket из Mac App Store
  2. Нажми плюсик и вставь ссылку подписки
  3. Выбери сервер и подключись

Где взять ссылку подписки?

Открой личный кабинет, выбери свою подписку и скопируй ссылку. Она выглядит примерно так:

https://api.frkn.org/subscription/abc123

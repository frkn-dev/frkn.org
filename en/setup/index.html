How to connect

Choose your platform and follow the instructions. Everything will be set up in a couple of minutes — just install the app and paste the subscription link.

iOS

iPhone and iPad

Streisand
Free Recommended
Download on the App Store
  1. Install Streisand from the App Store
  2. Open the app and tap Import
  3. Paste the subscription link from your personal account
  4. Select a server and tap Connect
Shadowrocket
Paid
Download on the App Store
  1. Buy and install Shadowrocket
  2. Tap the plus icon in the upper right corner
  3. Paste the subscription link
  4. Select a server and connect

Android

Smartphones and tablets

v2rayNG
Free Recommended
Download from GitHub
  1. Download and install the APK from GitHub
  2. Open the app and tap +
  3. Select Import from clipboard
  4. Paste the subscription link
  5. Select a server and tap the connection icon
Hiddify
Free
Go to website
  1. Install Hiddify from GitHub or Google Play
  2. Tap Add subscription
  3. Paste the subscription link
  4. Tap connect

Windows

PCs and laptops

v2rayN
Free Recommended
Download from GitHub
  1. Download and extract the archive from GitHub
  2. Run v2rayN.exe
  3. Click Servers → Import from clipboard
  4. Paste the subscription link
  5. Select a server and right-click → Set as active server
Hiddify
Free
Go to website
  1. Download and install Hiddify
  2. Click Add subscription
  3. Paste the subscription link
  4. Select VPN or proxy mode

macOS

MacBook and iMac

V2RayXS
Free Recommended
Download from GitHub
  1. Download the app and drag it to Applications
  2. Launch and open the menu from the status bar
  3. Click Import from clipboard
  4. Paste the subscription link
  5. Select a server and enable proxy
Clash Verge
Free
Download from GitHub
  1. Download and install Clash Verge
  2. Import the subscription link
  3. Select System Proxy or TUN mode
  4. Click the connect button
Shadowrocket
Paid
Download on the App Store
  1. Buy and install Shadowrocket from the Mac App Store
  2. Tap the plus icon and paste the subscription link
  3. Select a server and connect

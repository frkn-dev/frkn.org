FRKN Referral Program

How to get a referral code

Every active paid subscription has its own referral code. It is the last 12 characters of the subscription ID and can be found in the app / personal account.

Important: referral codes only work for paid subscriptions. If you are on a trial, you cannot invite friends or receive bonuses yet. Once you pay for a subscription, the code becomes active.

How to use a friend's code

Enter your friend's referral code when paying or activating a key. If the code is active:

  • you get your subscription as usual;
  • the friend who invited you gets bonus days.

How many days per referral

The number of bonus days depends on the promotion / settings. Usually it is X days added to the friend's current subscription.

Bonus conditions

The bonus is granted only if:

  1. The referral code belongs to an active paid subscription.
  2. You have completed a successful payment (or activated a paid key).
  3. You did not use your own code (self-referrals do not work).

Why the code may be invalid

  • The code owner's subscription has expired.
  • The code owner is still on a trial and has not paid yet.
  • The code was entered incorrectly.

Where to find your code

Your referral code is in your personal account — open the «Invite a friend» section to see your code and copy buttons.

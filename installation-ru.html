Установка

iOS | Android

Основная установка

  1. Установите приложение WireGuard для вашего устройства ( iOS | Android )
  2. Зайдите в установленное приложение
  3. В правом верхнем углу нажмите на +
  4. Выберите пункт "Создать из QR-кода"
  5. Наведите камеру телефона на сгенерированный QR-код на сайте
  6. Готово!

Установка из файла конфигурации

  1. Установите приложение WireGuard для вашего устройства ( iOS | Android )
  2. Скачайте файл конфигурации, нажав на кнопку "Скачать файл конфигурации"
  3. Зайдите в установленное приложение
  4. В правом верхнем углу нажмите на +
  5. Выберите пункт "Создать из файла или архива"
  6. Перейдите в папку, куда вы скачали файл конфигурации (он должен иметь расширение .conf) и выберите его
  7. Перключайте ползунок, чтобы включать/выключать VPN
  8. Готово!

MacOS | Windows

Установка из файла конфигурации

  1. Установите приложение WireGuard для вашего устройства ( MacOS | Windows )
  2. Скачайте файл конфигурации, нажав на кнопку "Скачать файл конфигурации"
  3. Зайдите в установленное приложение
  4. Нажмите на кнопку "Импорт туннелей из файла"
  5. Выберите файл конфигурации, который вы скачали с сайта
  6. Нажмите кнопку "Подключить"
  7. Готово!

Linux

Установка из файла конфигурации

  1. Перед установкой проверьте наличие доступных обновлений в вашей системе. При необходимости установите их: # Ubuntu и Debian sudo apt-get update && sudo apt-get upgrade
  2. Запустите установку WireGuard с помощью пакетного менеджера вашего дистрибутива: # Ubuntu и Debian sudo apt-get install wireguard
  3. Скачайте файл конфигурации, нажав на кнопку "Скачать файл конфигурации"
  4. Переместите скачанный конфиг в директорию /etc/wireguard/: sudo mv [путь до файла]/fuckrkn1.conf/etc/wireguard/
  5. Готово!
    1. Для установки соединения используйте команду wg-quick: sudo wg-quick up fuckrkn1.conf
      Ошибка /usr/bin/wg-quick: line 32: resolvconf: command not found:

      Если при попытке соединения выскочила эта ошибка — установите пакет resolvconf и повторите попытку:

      # Ubuntu и Debian sudo apt-get install resolvconf && sudo wg-quick up fuckrkn1.conf
    2. Для выяснения статуса подключения, используйте следующую команду: sudo wg show
    3. Для отключения соединения используйте wg-quick, но с атрибутом down: sudo wg-quick down fuckrkn1
